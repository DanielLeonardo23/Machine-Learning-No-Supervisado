DESCRIPCIÓN DEL PROYECTO
------------------------------------------------------

Este proyecto implementa un sistema de agrupamiento de usuarios mediante el algoritmo K-Means, usando un dataset real de comportamiento en redes sociales.

------------------------------------------------------
REQUISITOS CUMPLIDOS
------------------------------------------------------

✔ Dataset de Kaggle (train.csv con variables numéricas)
✔ Algoritmo de clustering: K-Means
✔ Permite ingresar el valor de K (editable en el notebook)
✔ Métrica de evaluación: Silhouette Score
✔ Visualización de clusters con reducción PCA a 2D
✔ Evidencia de interfaz: notebook interactivo con gráficos

------------------------------------------------------
ARCHIVOS INCLUIDOS
------------------------------------------------------

1. train.csv
   - Dataset original de usuarios en redes sociales

2. clustered_users.csv
   - Dataset con etiquetas de cluster aplicadas

3. kmeans_clustering.ipynb
   - Notebook interactivo con:
     • Limpieza de datos
     • Escalado
     • KMeans clustering
     • Métrica de evaluación
     • Visualización de clusters

4. cluster_plot.png
   - Imagen de dispersión PCA con clusters

5. README.txt
   - Este documento

------------------------------------------------------
INSTRUCCIONES DE USO
------------------------------------------------------

1. Abre el archivo `kmeans_clustering.ipynb` en Jupyter Notebook o Google Colab.
2. Cambia el valor de `k` en la celda correspondiente si deseas probar con otros clusters.
3. Ejecuta todas las celdas para ver el resultado del análisis y la gráfica.
4. Verifica la métrica `Silhouette Score` para evaluar la calidad de los clusters.
